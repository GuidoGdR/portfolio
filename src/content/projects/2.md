Este es un blog dinámico y totalmente responsivo para desarrolladores full-stack, basado en el robusto framework Django. La aplicación sirve como plataforma central para la autopromoción y la creación de contenido, demostrando competencia en la gestión de datos backend, el manejo seguro de formularios y diseño frontend.
<br>
Toda la pila de aplicaciones está en contenedores con Docker y Docker Compose para garantizar un entorno de desarrollo consistente y reproducible. La configuración incluye configuraciones independientes y optimizadas para simular entornos de producción, lo que facilita una implementación rápida.
